# GOJE TRAINING 🏋️‍♂️

**GOJE TRAINING** is a premium, high-precision workout timer designed for elite training. It combines a versatile interval timer with a millisecond-accurate stopwatch in a modern, immersive, and professional interface.

## ✨ Elite Features

- **Dynamic Workout Timeline:** Create custom sequences with unique work and rest durations.
- **Drag & Drop Reordering:** Reorder your sets easily using a mobile-first, sortable list.
- **Batch Generator:** Instantly generate multiple work/rest sets for high-intensity training.
- **Voice Guidance (TTS):** Premium English text-to-speech voice guidance that motivates you through every phase.
- **Energetic Motivational Cues:** "Sweat is just fat crying!", "Stay hungry!", "Don't stop now!"—over 40 energetic phrases to keep you moving.
- **Distance Mode:** An immersive, color-coded background (Red for Work, Blue for Rest) that pulses with your heart rate and is visible from across the room.
- **Automatic Orientation Lock:** Automatically locks to **Landscape Mode** when a workout starts for maximum visibility on any screen.
- **Smart Library:** Save and load your favorite workout routines locally on your device.
- **High-Precision Stopwatch:** Millisecond-level accuracy with lap tracking and responsive animations.
- **Screen Wake Lock API:** Keeps your device screen active throughout your training.
- **PWA & Native Mobile Ready:** Fully responsive design with **Capacitor** integration for Android and iOS.

## 📱 Mobile App (Android & iOS)

GOJE is now a cross-platform app! 

### **Android & iOS Commands**
We use **Capacitor** to sync our web project with native mobile folders.

```bash
# Build and Sync with Android
npm run android

# Build and Sync with iOS
npm run ios

# Sync only (after code changes)
npm run cap:sync
```

## 🛠️ Tech Stack

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router & Static Export)
- **Native:** [Capacitor 8](https://capacitorjs.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Drag & Drop:** [@dnd-kit](https://dndkit.com/)
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🏃 Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open [http://localhost:3000](http://localhost:3000)** in your browser or mobile device.
