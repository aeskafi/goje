# GOJE TRAINING

**GOJE TRAINING** is a premium, high-precision workout timer designed for elite training. It combines a versatile interval timer with a millisecond-accurate stopwatch in a modern, professional interface.

## 🚀 Features

- **Dynamic Workout Timeline:** Create custom sequences with unique work and rest durations for every set. No more being restricted by fixed patterns.
- **High-Precision Stopwatch:** Built using `requestAnimationFrame` and `Date.now()` for millisecond-level accuracy with lap tracking.
- **Screen Wake Lock API:** Keeps your device screen active during intense workouts so the timer never pauses or dims.
- **Premium UI/UX:** A glassmorphism design powered by **Framer Motion** for smooth, fluid transitions and an immersive training experience.
- **PWA Ready:** Install it on your iOS or Android home screen for a native app feel.
- **Web Audio Cues:** Clear, loud audio signals for interval transitions.

## 🛠️ Built With

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 📜 Original Project Credits

This project is a modernized and enhanced port of the original [movimiento-training-timer](https://github.com/movimiento23/movimiento-training-timer). 

We've evolved the original concept from a vanilla HTML/JS Spanish-language timer into a high-performance React application with enhanced timing precision, English localization, and mobile-first features like the Wake Lock API.

## 🏃 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to start training.

## 📱 Installation

To install as a PWA:
1. Open the app in your mobile browser.
2. Tap the "Share" button (iOS) or "Menu" icon (Android).
3. Select **"Add to Home Screen"**.
